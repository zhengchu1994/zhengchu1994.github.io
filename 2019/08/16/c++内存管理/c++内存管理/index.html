内存分配方式

（1） 静态存储区域分配。编译的时候已经分配好，如static变量。

（2）栈上创建的内存，如函数内的局部变量。

（3）堆上分配的动态内存。如运行时用malloc或者new申请的内存，这时候程序员负责使用free或delete销毁。

防止内存报错的几个规则

（1）用malloc或new申请内存后，立即检查是否为nullptr;

（2）为数组或动态内存赋初值，防止违背初始化的内存作为右值使用;

（3）避免数组或指针的下标越界；

（4）动态内存申请与释放必须配对，方式内存泄露；

（5）用free或delete释放内存后，立即将指针设为nullptr，防止出现野指针。
